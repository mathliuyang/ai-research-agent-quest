# Case 05: Vibe Coding + Vibe Research

这个案例替换原来的“网页 / 小工具”示例，展示一次真实的 AI+科研流程：

1. 读取论文 PDF，定位算例 1 的方程、参数、误差定义和原文图表。
2. 用 Python 实现 GBDF2-IMEX 复现代码，并生成时间/空间误差曲线。
3. 对比原文截图和复现结果，核验二阶收敛结论。
4. 生成中文 HTML 对比报告，便于现场展示和后续复用。
